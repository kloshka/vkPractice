# Ручной Smoke Checklist (MVP)

Дата последнего прогона: 2026-04-17

## Как использовать

1. Запустить приложение.
2. Пройти кейсы по шагам в двух браузерных сессиях (обычное окно + инкогнито).
3. Отмечать фактический результат в колонке Статус.
