# Отчет по тестированию MVP

Дата: 2026-04-17

## 1) Smoke checklist (15 кейсов)

Команда запуска:
- npm run test:smoke

Итог:
- PASS 15/15

Покрыто:
- регистрация/авторизация
- разграничение ролей
- создание квиза и вопросов
- запуск комнаты и подключение по коду
- realtime-старт, ответ в активное окно
- блокировка двойной отправки
- история у обеих ролей

## 2) Нагрузочный мини-тест

Команда запуска:
- $env:PARTICIPANTS=30; npm run test:load

Итог:
- LOAD PASSED
- Participants: 30
- Joined: 30
- Leaderboard size: 30
- Score range: 1..3
- Duration ms: 15123

Вывод:
- Сценарий на верхней границе требования (30 участников) отработал успешно.

## 3) Граничные случаи

Команда запуска:
- npm run test:edge

Итог:
- PASS 4/4

Проверено:
- пустые поля (валидация)
- повторные входы одним пользователем
- двойной клик по отправке ответа
- обрыв соединения и повторное подключение в live-сессии

## 4) Базовый e2e-сценарий

Команда запуска:
- npm run test:e2e

Итог:
- E2E PASSED
- Final scores: Alice 2, Bob 1, Cara 0

## 5) Общий вывод

По результатам автоматических прогонов MVP работоспособен по ключевым требованиям ТЗ и выдерживает тестовую нагрузку до 30 одновременных участников в рамках текущего сценария.
