# Краткая пояснительная записка (MVP)

## 1. Цель проекта

Разработать веб-приложение для проведения квизов в реальном времени с ролями организатора и участника.

## 2. Выбор средств разработки

- **Node.js + Express**: быстрый серверный API для MVP
- **SQLite**: простая локальная БД с реляционной моделью
- **Socket.IO**: real-time синхронизация вопросов и результатов
- **HTML/CSS/JS**: легкий фронтенд без сложной сборки

## 3. Основные этапы разработки

1. Спроектирована модель БД: пользователи, квизы, вопросы, сессии, ответы, очки
2. Реализована авторизация и разграничение ролей
3. Реализованы CRUD-операции организатора для создания квизов и вопросов
4. Добавлен запуск комнаты с уникальным кодом
5. Настроен WebSocket-поток:
   - подключение участников
   - показ текущего вопроса по таймеру
   - прием ответов только в активный интервал
6. Добавлен подсчет очков и финальный лидерборд
7. Добавлен личный кабинет и история сессий

## 4. Что реализовано в MVP

- создание и запуск квиза организатором
- подключение участника по коду
- вопросы с одиночным и множественным выбором
- текстовые и с изображением
- real-time синхронизация
- лидерборд и сохранение результатов
